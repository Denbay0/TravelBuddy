## About

**TravelBuddy** — full-stack веб-приложение для путешественников.

Приложение объединяет **планирование маршрутов** и **социальную платформу**, где пользователи могут делиться опытом путешествий, публиковать посты и находить новые идеи для поездок.

---

## Features

- **Routes**  
  создание и редактирование маршрутов, список городов, длительность, транспорт, trending маршруты и избранное  

- **Posts**  
  публикации о путешествиях, лайки, сохранение постов, комментарии  

- **Profiles**  
  профиль пользователя, аватар, био, список своих постов и сохранённых маршрутов  

- **Search**  
  глобальный поиск по маршрутам, постам и пользователям  

- **PDF Reports**  
  генерация PDF-отчётов по маршрутам  

---

## Wired

Иногда кажется, что путешествия — это просто точки на карте.  
Но на самом деле это **следы**, которые остаются в людях и историях.

TravelBuddy — это место, где маршруты превращаются в опыт,  
а опыт — в истории, которые продолжают жить в **wired**.
